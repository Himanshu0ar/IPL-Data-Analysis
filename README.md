# IPL Data Analysis Project 🏏

This project analyzes IPL (Indian Premier League) match data using Python libraries such as Pandas, NumPy, and Matplotlib.

The goal of this project is to extract useful insights from IPL datasets and visualize team performances, player statistics, venue analysis, and season-wise trends.

---

## 📌 Features

- Most Successful IPL Teams
- Toss Winner Analysis
- Top Run Scorers
- Top Sixes and Fours
- Venue Analysis
- Win Percentage Analysis
- Season Wise Analysis

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

---

## 📊 Dataset Used

- matches.csv
- deliveries.csv

---

## 📈 Key Insights

- Mumbai Indians are among the most successful IPL teams.
- Virat Kohli is one of the top IPL run scorers.
- The number of IPL matches has increased over the years.
- Toss results can influence match outcomes.
